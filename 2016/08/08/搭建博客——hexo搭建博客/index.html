发表于 2016-08-08

什么是 Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装前提

安装 Hexo 相当简单。然而在安装前，电脑中必须安装了下列应用程序：

  • Node.js
  • Git

正式安装

安装

  • 安装 Git

Windows：git-2.9.2 更新时间:2016.7.16 下载地址

Mac： git-2.7.1 更新时间:2016.7.8 下载地址

Linux：sudo apt-get install git-core

  • 安装 Node.js

方式一：下载 安装程序 来安装 Node.js。

方式二：使用 nvm 安装（最佳方式）。

cURL: $ curl https://raw.github.com/creationix/nvm/master/install.sh | sh

Wget: $ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh

安装完成后，重启终端并执行下列命令即可安装 Node.js。
$ nvm install stable

  • 安装 Hexo

所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。

$ npm install -g hexo-cli

初始化

mac系统打开命令行控制台，默认根目录是user,个人建议最好新建一个子目录，然后初始化安装。

本地启动

启动本地服务，进行文章预览调试，命令：

浏览器输入 http://localhost:4000 进行效果预览。
